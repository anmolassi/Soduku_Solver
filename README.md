# Soduku_Solver
This program helps you to solve 3X3 soduko.
Just put 0 in place of blank.
Provide a space between numbers while input.
Sample testcase:
0 7 0 0 0 9 0 3 0
6 9 0 0 0 1 0 7 0
1 0 8 7 0 0 0 0 0
0 0 2 0 0 0 0 0 0
0 0 0 8 0 0 0 0 6
0 0 0 5 2 0 0 0 9
0 6 0 0 0 0 5 0 0
8 0 0 9 0 7 0 0 0
7 0 0 2 0 0 6 0 0
